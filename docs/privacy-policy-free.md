# 카메라 셔터 무음 (무료) — 개인정보처리방침

**최종 업데이트:** 2026년 7월 5일

> 이 방침은 **무료(광고) 앱** (`com.mute.shutter.free`)용입니다.  
> 유료 버전은 [`privacy-policy.md`](privacy-policy.md)를 참고하세요.

## 개요

카메라 셔터 무음 (무료)(이하 "앱")은 사용자 기기에서 동작하며, Google AdMob을 통해 광고를 표시합니다.

## 앱이 하는 일

- 무선 디버깅(ADB)으로 삼성 갤럭시 카메라 셔터음 설정을 변경합니다.
- 카메라 앱 사용 중에만 일시적으로 기기 볼륨을 조절합니다.
- Google AdMob을 통해 배너·전면 광고를 표시합니다.

## 권한 사용

| 권한 | 목적 |
|------|------|
| 인터넷 / Wi‑Fi | 기기 내 무선 ADB 연결, AdMob 광고 로드 |
| 사용 통계 | 카메라 앱 실행 감지 (외부 전송 없음) |
| 알림 | 백그라운드 카메라 감시 서비스 표시 |
| 부팅 완료 | 재부팅 후 설정 자동 재적용 |
| AD_ID | AdMob 광고 식별 (Google Play 서비스) |

## 데이터 수집

앱 자체는 다음을 **직접 수집하지 않습니다:**
- 이름, 이메일, 전화번호
- 위치 정보
- 사진·영상

설정값(페어링 여부, 연결 포트)은 **기기 내부에만** 저장됩니다.

### AdMob (Google)

앱에 포함된 Google AdMob SDK가 광고 제공을 위해 다음을 수집할 수 있습니다:
- 광고 ID (Advertising ID)
- 기기 정보, IP 주소
- 광고 상호작용 데이터

자세한 내용: [Google 광고 정책](https://policies.google.com/technologies/ads)

## 제3자 공유

- **Google AdMob** — 광고 표시 및 측정
