# 카메라 셔터 무음 — 개인정보처리방침

**최종 업데이트:** 2026년 7월 5일

> 이 방침은 **유료 앱** (`com.mute.shutter`)용입니다.  
> 무료(광고) 버전은 [`privacy-policy-free.md`](privacy-policy-free.md)를 참고하세요.

## 개요

카메라 셔터 무음(이하 "앱")은 사용자 기기에서만 동작하며, 개인정보를 수집·전송·저장하지 않습니다.

## 앱이 하는 일

- 무선 디버깅(ADB)으로 삼성 갤럭시 카메라 셔터음 설정을 변경합니다.
- 카메라 앱 사용 중에만 일시적으로 기기 볼륨을 조절합니다.

## 권한 사용

| 권한 | 목적 |
|------|------|
| 인터넷 / Wi‑Fi | 기기 내 무선 ADB 연결 |
| 사용 통계 | 카메라 앱 실행 감지 (외부 전송 없음) |
| 알림 | 백그라운드 카메라 감시 서비스 표시 |
| 부팅 완료 | 재부팅 후 설정 자동 재적용 |

## 데이터 수집

앱은 다음을 **수집하지 않습니다:**
- 이름, 이메일, 전화번호
- 위치 정보
- 사진·영상
- 광고 ID

설정값(페어링 여부, 연결 포트)은 **기기 내부에만** 저장됩니다.

## 제3자 공유

없음. 광고 SDK를 사용하지 않습니다.

## 문의

Play Store 앱 페이지의 개발자 연락처를 이용해 주세요.
