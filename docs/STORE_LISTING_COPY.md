# ① 유료 앱 (`com.mute.shutter`)

## 앱 이름 (30자 이내)
```
카메라 셔터 무음
```

## 간단한 설명 (80자 이내)
```
삼성 갤럭시 카메라 셔터음 끄기. 개발자 옵션·무선 디버깅 1회만. PC 불필요, 광고 없음.
```
(49자)

## 자세한 설명 (4000자 이내)

```
카메라 셔터 무음은 삼성 갤럭시(S25, S26 Ultra 등)에서 카메라 촬영음(셔터음)을 PC 없이 끄는 유료 앱입니다. 무선 디버깅을 한 번만 설정하면 이후에는 앱을 열기만 해도 자동으로 다시 적용됩니다.

■ 이 앱이 하는 일
• 삼성 갤럭시 카메라 셔터음 설정을 무음으로 변경
• 카메라 앱 사용 중 촬영음 자동 무음
• 앱 실행·재부팅 후 설정 자동 재적용 시도
• 광고 없음 — 깔끔한 화면

■ 필요한 것
• 삼성 갤럭시 스마트폰
• Android 13 이상 (64비트)
• Wi-Fi 연결 (무선 디버깅용)
• 약 3~5분 (최초 1회만)

━━━━━━━━━━━━━━━━━━━━
■ 설정 방법 (처음 한 번만)
━━━━━━━━━━━━━━━━━━━━

【1단계】 개발자 옵션 켜기

개발자 옵션은 처음에 숨겨져 있습니다. 아래 순서대로 켜 주세요.

① 「설정」 앱을 엽니다.

② 아래로 스크롤해 「휴대전화 정보」 또는 「디바이스 케어 및 배터리」→「휴대전화 정보」를 탭합니다.
   ※ 기종·One UI 버전에 따라 「디바이스 정보」「휴대전화 정보」 등 이름이 조금 다를 수 있습니다.

③ 「소프트웨어 정보」를 탭합니다.

④ 「빌드 번호」를 손가락으로 **7번** 빠르게 연속 탭합니다.
   • 탭할 때마다 "N번 더 탭하면 개발자가 됩니다" 같은 안내가 나옵니다.
   • 7번째 탭 후 "개발자 모드가 켜졌습니다" (또는 비슷한 문구)가 표시되면 성공입니다.

⑤ 화면 잠금(PIN·비밀번호·지문)을 요구하면 입력합니다.

⑥ 설정 화면 맨 위 또는 검색창에 「개발자 옵션」을 검색해 들어갑니다.
   • 「설정」→「개발자 옵션」 메뉴가 새로 생겼는지 확인하세요.

【2단계】 무선 디버깅 켜기

① 「설정」→「개발자 옵션」으로 이동합니다.

② 목록에서 「무선 디버깅」을 찾습니다.
   • 검색창에 "무선 디버깅" 입력하면 빠릅니다.

③ 「무선 디버깅」 오른쪽 스위치를 **켬(ON)** 으로 바꿉니다.
   • 처음 켤 때 확인 팝업이 나오면 「허용」 또는 「확인」을 누릅니다.

④ ⚠️ 중요: 스위치만 켜고 끝내지 마세요.
   「무선 디버깅」 **글자(항목 이름)를 탭**해서 안쪽 메뉴로 들어갑니다.

【3단계】 페어링 코드 확인

① 무선 디버깅 상세 화면에서 「페어링 코드로 기기 페어링」을 탭합니다.

② 「Wi-Fi 페어링 코드」 팝업(또는 전체 화면)이 열리면 아래 두 가지를 확인합니다.
   • **Wi-Fi 페어링 포트** — 숫자 5자리 (예: 37841)
   • **페어링 코드** — 숫자 6자리 (예: 123456)

③ 이 화면을 **닫지 말고** 그대로 두세요. (코드는 약 1~2분 후 만료됩니다)

【4단계】 앱에서 입력

① 「카메라 셔터 무음」 앱을 실행합니다.

② 화면에 보이는 입력란에 방금 확인한 값을 넣습니다.
   • 「페어링 포트」 ← Wi-Fi 페어링 포트 숫자
   • 「6자리 PIN」 ← 페어링 코드 6자리

③ 「처음 설정하기」 버튼을 누릅니다.

④ 잠시 기다리면 상태가 「셔터음 꺼짐」으로 바뀝니다.
   • PIN이 만료됐다면 3단계부터 다시 하세요. (새 코드 발급)

【5단계】 (필요할 때만) 사용 통계 허용

일부 기종(S26 Ultra 등)에서는 「사용 통계 허용」 버튼이 표시될 수 있습니다.

① 앱에서 「사용 통계 허용」 버튼을 탭합니다.

② 설정 화면으로 이동하면 「카메라 셔터 무음」을 찾아 **허용**으로 바꿉니다.
   • 경로: 설정 → 앱 → (특수 액세스) → 사용 정보 접근
   • 또는 설정 검색창에 "사용 정보 접근" 검색

③ 앱으로 돌아와 「셔터음 끄기」 또는 「다시 적용」을 한 번 더 누릅니다.

━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━

• 앱만 열면 저장된 설정으로 자동 재연결·재적용됩니다.

• 「고급 설정」→「처음부터 다시 설정」으로 페어링을 초기화할 수 있습니다.

Q. 「설정 필요」만 보여요.
→ 무선 디버깅이 꺼져 있지 않은지, 개발자 옵션이 켜져 있는지 확인하세요.

Q. PC가 필요한가요?
→ 아니요. 폰만으로 설정합니다.

Q. 무료 버전이 있나요?
→ Play Store에서 「카메라 셔터 무음 (무료)」를 검색하세요. (광고 포함)

■ 유료 버전 장점

• 개인정보·사진·연락처 등을 수집하지 않습니다.
• 문의: Play Store 개발자 연락처
```
