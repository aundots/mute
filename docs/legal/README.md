# 개인정보처리방침 URL (Play Console)

GitHub Pages 활성화 후 아래 URL을 Play Console에 등록하세요.

| 앱 | Play Console URL |
|----|------------------|
| **유료** (`com.mute.shutter`) | `https://aundots.github.io/mute/legal/privacy.html` |
| **무료** (`com.mute.shutter.free`) | `https://aundots.github.io/mute/legal/privacy-free.html` |

## GitHub Pages 설정 (1회)

1. GitHub에 `aundots/mute` 저장소 생성 후 이 프로젝트 push
2. **Settings** → **Pages**
3. Source: **Deploy from a branch** → Branch: **main** / Folder: **/docs**
4. Save → 1~2분 후 위 URL 접속 확인

## 다른 호스팅

- **Vercel / Netlify**: `docs/legal/privacy.html` 을 정적 호스팅해도 됩니다.
- **아이와함께 Vercel**에 경로 추가는 권장하지 않습니다 (앱별 URL 분리).
